# DummyApi
Тестирование проекта DummyApi с помощью Postman.

Объект - **Post** (пост).

Действия: **Get Post List** (получение списка постов), **Create Post** (создание поста).

Процесс тестирования:

- Анализ технической документации.
- Составление майнд-карты.
- Составление запросов.
- Тестирование.
  
Также составлены автотесты для объекта **Post**.

Запросы: **Get Post List, Create Post, Get Post by ID, Update Post, Delete Post**.

## Оглавление

## Описание проекта
https://dummyapi.io/ - это сервис для тестирования API. Для выполнения запросов необходим app-idd, который можно получить после регистрации на сайте. В качестве тестирования был взят объект Post.

----

### Post
#### GET /post (Get List)
Возвращает список постов, отсортированных по дате создания.

* доступен queary params для вывода определенной страницы.
* доступен queary params для вывода постов созданных в текущей среде.
  
**Response body:**

**List**
```javascript
{
data: Array(Model)
total: number(total items in DB)
page: number(current page)
limit: number(number of items on page)
}
```
**Post Previev**
```javascript
{
id: string(autogenerated)
text: string(length: 6-50, preview only)
image: string(url)
likes: number(init value: 0)
tags: array(string)
publishDate: string(autogenerated)
owner: object(User Preview)
}
```
**User Previev**
```javascript
{
id: string(autogenerated)
title: string("mr", "ms", "mrs", "miss", "dr", "")
firstName: string(length: 2-50)
lastName: string(length: 2-50)
picture: string(url)
}
```

----

**POST /post/create (Create Post)**
Создает новый пост, возвращает данные созданного поста.

**Request body:**

**Post create**
```javascript
{
text: string(length: 6-50, preview only)
image: string(url)
likes: number(init value: 0)
tags: array(string)
owner: string(User id)
}
```
**Response body:**

**Post**
```javascript
{
id: string(autogenerated)
text: string(length: 6-1000)
image: string(url)
likes: number(init value: 0)
link: string(url, length: 6-200)
tags: array(string)
publishDate: string(autogenerated)
owner: object(User Preview)
}
```

## Автотесты
Для объекта **Post** созданы автотесты для следующих запросов: (GET) Get Post List, (POST) Create Post, (GET) Get Post by ID, (PUT) Get Post List (update), (DEL) Delete Post by ID. 
Автотесты включают в себя проверки на статус-код ответа, время ответа сервера, соответствие ключей и значений.
Для запуска автотеста нужно, чтобы все запросы были выбраны и стояли в следующем порядке: (GET) Get Post List, (POST) Create Post, (GET) Get Post by ID, (PUT) Get Post List, (DEL) Delete Post by ID. 
Автотесты можно посмотреть [тут](https://github.com/jsinabdeeva/DummyApi/blob/main/DummyAPI%20(autotest).postman_collection.json).
